Tabulation* Tabulation_new ( double  windowcm,
int  nbins 
)

Allocate a new object of type Tabulation.

This is accomplished via a single call to malloc, so that the entire object will reside in contiguous memory. This minimizes page faults and should make time-critical functions such as Tabulation_record run faster.

Parameters:
[in]windowcmThe size in centimorgans of the window used to scan the genome.
[in]nbinsThe number of bins in which to tabulate values.
Returns:
A newly allocated object of type Tabulation, which should be freed with Tabulation_free.
