int Hill_evolveDiscrete ( double  y[],
PopHist *  ph,
double  c,
double  u 
)

Evolve through all Epochs of population history, using step size h and beginning with initial equilibrium.

Calculation iterates Hill's difference equation. The initial value of y is set using Hill's equation for equilibrium.

Parameters
[out]yOn return, y contains Hill's vector of moments.
[in]phDescribes the population's history. If ph contains just one Epoch, y gets its equilibrium value.
[in]cRecombination rate; overrides values in ph.
[in]uMutation rate.
Returns
Always returns 0.
