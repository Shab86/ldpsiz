PopHist* PopHist_perturbInPlace ( PopHist *  ph,
double  dt,
double  dNinv,
gsl_rng *  rng 
)

Perturb parameters in PopHist structure ph.

Each parameter is perturbed away from its initial value. The distribution of the perturbations depends on the macros PERTURB_GAUSSIAN and PERTURB_TDIST. If the first of these is defined at compile time, then perturbations are Gaussian. Otherwise, if the second macro is defined, perturbations are drawn from a t distribution. Otherwise they are uniform.

1/2N values are reflected back and forth so that the perturbed value lies within [loTwoNinv, hiTwoNinv].

Parameters
[in,out]phThe PopHist to be perturbed.
[in]dtcontrols the magnitude of perturbations in the duration variable.
[in]dNinvcontrols the magnitude of perturbations in 1/2N.
[in]rngrandom number generator.
Returns
ph
