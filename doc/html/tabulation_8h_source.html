00016  * tabulation.h. This file defines the Tabulation object, which keeps
00017  * track of comparisons between pairs of SNPs. These comparisons take
00018  * place within a window of size windocm centimorgans, which slides
00019  * across the genome. Comparisons are tabulated into nbins bins, based
00020  * on the distance that separates them along the chromosome. The
00021  * Tabulation object represents a table, with a row for each bin, and
00022  * a column for each of: nobs, sep_cm, numerator, denominator. Each
00023  * time a pair of SNPs is compared, we add something to each of these
00024  * columns in the row determined by the distance that separates the
00025  * SNPs along the chromosome.
00026  *
00027  * The definition of struct Tabulation is exposed here (rather than
00028  * hidden in tabulation.c) for the benefit of the inline functions
00029  * defined at the bottom of this file.
