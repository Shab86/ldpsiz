long ThreadArg::sampling_interval

For production, set sampling_interval=1.

For debugging, set to (say) 5 to process only every 5th SNP. This makes execution faster.

ThreadBounds* ThreadArg::tb

In a multithreaded environment, each thread operates only on a portion of the chromosome.

The Threadbounds object, tb, specifies the SNPs to be considered by the current thread.
